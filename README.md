# 🦁 RawMage (生肉狮) - The Ultimate AI-Powered ACG Localization Platform

[![Website](https://img.shields.io/badge/Official_Website-my--lion--bot.online-2997ff?style=for-the-badge&logo=google-chrome)](https://my-lion-bot.online/)
[![Telegram Bot](https://img.shields.io/badge/Telegram-RawMage_Bot-2CA5E0?style=for-the-badge&logo=telegram)](https://t.me/ZHCartoon_bot)

**RawMage (Chinese: 生肉狮)** is a zero-download, Telegram-native AI translation SaaS and bot tailored for ACG (Anime, Comic, Games) creators, game localizers, and scanlation groups. It provides automated high-accuracy translation, typesetting, and project export (PSD) for raw manga, webtoons, light novels, and game UI text.

## 🚀 Core Features & UVP (Unique Value Proposition)
* **Zero-Download & Telegram-Native:** Operates entirely within Telegram. The workflow is driven by highly intuitive dynamic button clicks.
* **Dual-Layer Glossary System:** Built-in Personal Glossaries and Global System Glossaries for specialized ACG/fandom terminology control.
* **Ultimate Customization:** Users can fully configure their preferred Translation Engines (GPT-4o, Gemini 2.5, Qwen), OCR Technologies, Text Erasure Models, and Layout Formats.
* **Advanced Game Localization:** Accurate parsing and translation of game language data files. Fully supports Ren'Py and Unity game engines.
* **Link-to-Translation:** Submit raw content URLs to receive localized text automatically.
* **Reverse Image Source Tracing:** Built-in reverse image lookups to instantly locate original raw comic sources.
